# Personal Finance Dashboard

A Streamlit-based personal finance application that helps users track expenses, analyze spending patterns, and understand their financial health.

## Features
- Upload and manage transaction data
- Category-wise spending analysis
- Budget comparison and savings insights
- Scenario-based spending simulator

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
